# Arpit Sangal — Airtel-Themed Portfolio

A dedicated copy of the portfolio styled to match [Airtel Careers](https://careers.airtel.com/) — Airtel red, lavender sections, corporate navigation, and blob branding.

## Quick Start

```bash
cd arpit-portfolio-airtel
npm install --legacy-peer-deps
npm start
```

## Deploy

```bash
npm run deploy
```

Live URL (after deploy): **https://arpitsangal1997.github.io/arpit-portfolio-airtel/**

> Note: GitHub Pages allows one site per repo. Use a separate repo or branch for the Airtel variant, or deploy manually with `npm run deploy` to the `gh-pages` branch under a subpath.
